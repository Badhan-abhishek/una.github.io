This blog post assumes that you’ve never set up any kind of server environment on your system before and have no idea how to get started. It’s not the only way, but its an inclusive set of directions to follow. If you’ve already read the Craft Installation Docs, and were confused, then this post is for you.

  1. First you’ll want to install a program like MAMP or XAMPP. These programs basically bundle install Apache, MySQL, and PHP on your system, and make it easy to access them with a GUI that gives you start and stop buttons.
  2. Install Sequel ProÂ to help you manage your MySQL databases.
  3. Open MAMP (not MAMP Pro!) and click the “Start Servers” button.



  4. In MAMP, click “Open Start Page” button right below “Start Servers”. The first chart on this page will show your your login parameters
  5. Open Sequel Pro and use these parameters to open a connection. Your host should be “localhost” or “127.0.0.1” and your use name and password are probably both “root.”


  6. After hitting “Connect,” create a new database by clicking Database > Add Database, or option + shift + A. Name your database something that makes sense to your project.
  7. Go to buildwithcraft.com and install the latest version of Craft.
  8. Unzip the file and place it in the httdocs folder inside of the MAMP Application.
  9. Navigate to craft/config/db.php and open the db.php file in any text editor.
  10. Edit this file with your MySQL and database information. If you used MAMP, yours will look similar to this example, except for the database name: 
    <?php 

return array(

/**
 * The database server name or IP address.
 */
'server' => '127.0.0.1',

/**
 * The database username to connect with.
 */
'user' => 'root',

/**
 * The database password to connect with.
 */
'password' => 'root',

/**
 * The name of the database to select.
 */
'database' => 'unicorns',

/**
 * The prefix to use when naming tables. This can be no more than 5 characters.
 */
'tablePrefix' => 'craft',

);
  11. Rename your “htaccess” file to “.htaccess” (just add a “.” in front of it) and delete the web config file. This can be tricky, as your computer may complain. If it says that this is a reserved name, make sure to go to enable hidden files. To do this, open your Terminal and type in: 
    defaults write com.apple.Finder AppleShowAllFiles YES
    Restart finder and try to rename the file again.
  12. At this point, if you followed those instructions directly, you should be able to go to localhost/Craft/public/admin or 127.0.0.1/Craft/public/admin in your web browser. This is how you will access the admin panel later.
  13. From here, press the big Install button and let Craft take you through the setup process
  14. Your (public-facing) website should be accessible at localhost/Craft/public or 127.0.0.1/Craft/public/
